PAIRS MEMORY GAME


Pairs Memory Game is an Android application developed within "Android Studio", as part of Assessed Coursework.

The base game mechanics follow that of a typical "Pair Match" game; the player selected a facedown card and picks a second one in attempts to match the initally selected card.

The application funcationality extened to that of which a Puzzle list is acquired through dedicated ".json" file located on a server, when the app has an internet connection the puzzle list can be acquired, with each puzzle generating appropriately based on the data from the selected puzzle. Puzzles are able to be saved/stored locally to allow play offline. Add-ons and Puzzle updates can be made to the server, which are automatically adapted to within the app.
