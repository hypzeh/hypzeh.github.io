TIDE BOX


Tide Box project is set in a professional environement, working as a team of 6 to develop an application to simulate flooding with VR capabilities.

Humber in a Box was a project developed by previous SEED Software students in 2014/2015. The purpose of which was to simulate a flood from the Humber, which was proven to be successful at numerous public events used to demonstrate the product.

The client seeks the next iteration of the application, under the name Tide Box. The Tide Box project will update Humber in a Box for use with the latest Oculus Rift headsets. Further updates will include quality of life upgrades, making it more intuitive for users, allowing them to navigate the application through series of scenarios without the need for a guide. Additional functionality as a desktop application for use with touchscreens, and support for VISR cardboard and mobile VR are all possible additions to the project. Humber in a Box was created using Unity Engine, the Tide Box project will be developed using Unreal Engine.
