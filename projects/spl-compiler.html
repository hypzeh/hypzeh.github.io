SPL COMPILER


SPL Compiler was assigned in the third year of Computer Science, the aim of which was to create a fully working compiler for a small simple programming language called SPL. The compiler is able to read in SPL source code and produce ANSI C as output.

A description of the language was provided, from which a "BNF" roadmap document was created. Using research material and tools such as "Flex" and "Bison" the project consisted of a Lexical Analyser and a Parser.

Additionally, the Parser contains all of the code necessary to create a parse tree using the Yacc and Bison Tools. This was demonstrated using a print tree capability of which could be enabled by the C macro DEBUG; saving the results in a formatted output file.
