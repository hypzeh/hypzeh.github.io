A FAULT IN OUR TREES


A Fault in Our Trees is a JavaScript project assigned as Assessed Coursework in my second year at University of Hull.

The concept being to create an algorithm to draw a Fault Tree, based on the data provided. The data is a deductive failure analysis in which undesired states of a system get triggered; these states are placed into a generalised node and will be displayed in a tree format. There are 3 sets of data, each increasing in number of faults, the end result should be to demonstrate an adaptive tree; meaning it isn't just tied down to the 3 data sets and can be adaptively drawn/generated based on the data provided.

Further functionality includes, zoom, pan, mini-map and a guide.
