SANDBOX TOOL


Windows Sandbox Tool was developed as a research piece into the concepts of trustworthy computing. This tool permits an executable program to be run with reduced access permissions; use case scenario being able to run student coursework submissions during assessment without allowing any form of unwanted/malicious code to be executed.

The tool also handles command-line arguments; in both aspects of being operated and executing programs. This allows the execution of programs that take in parameters with desired restrictions, while also operating without the User Interface and through a script given through a command prompt.

Additional restriction to this tool can be implemented through a very simple process of adding the desired permission namespace. Any permission requests will be displayed to the user, for them to act upon if necessary.
