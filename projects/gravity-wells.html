GRAVITY WELLS


The aim of the ACW is to research, design and implement a distributed, physically-based modelling demonstrator, for a series of gravity wells. The world consists of a large, planar surface, surrounded by a circular wall. A large number of spheres (balls) are placed on the surface. The simulation is run across a number of peers, each displaying a consistent view of the world. The simulation allows a user to apply a force to the balls.
